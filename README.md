🚀 JobPilot: Your AI-Powered Career Co-Pilot

JobPilot is a next-generation career assistant designed to automate the most tedious parts of the job search. Built with the latest Next.js 15, Google Gemini AI, and Supabase, it helps you analyze job descriptions, track applications via a Kanban board, and generate personalized referrals in seconds.

## ✨ Key Features

### 🤖 AI Job Analyzer
Paste a job URL or description to get an instant match score.  
Extracts requirements using Puppeteer and evaluates your resume using Gemini AI.

### 📋 Smart Kanban Board
Manage your job applications with a sleek drag-and-drop interface powered by `@hello-pangea/dnd`.

### ✉️ AI Referral Generator
Generate personalized LinkedIn and Email outreach messages tailored to job roles and company context.

### 🛡️ Intelligent Rate Limiting
Implements server-side usage tracking using Supabase RLS and IP-based gating to manage AI token consumption.

### 🎨 Ultra-Responsive UI
Built with Tailwind CSS and Framer Motion for a smooth, modern experience.

---

## ⚙️ Tech Stack

- **Frontend:** Next.js (App Router), React, Tailwind CSS, Framer Motion  
- **Backend:** Node.js, REST APIs  
- **Database & Auth:** Supabase (PostgreSQL, RLS)  
- **AI:** Google Generative AI (Gemini)  
- **State Management:** Zustand  
- **Web Scraping:** Puppeteer, Cheerio  
- **Parsing:** PDF.js, Mammoth  

---

## 💡 How It Works

1. **Upload Resume**  
   Your resume is securely parsed and stored.

2. **Analyze Jobs**  
   Paste a LinkedIn or Indeed URL. JobPilot extracts and compares it with your profile.

3. **Get Insights**  
   Identify skill gaps, receive suggestions, and view a readiness score.

4. **Track & Network**  
   Manage applications via dashboard and generate referral messages.
